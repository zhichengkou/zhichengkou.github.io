强化学习中Q矩阵计算方法的总结

发表于 2025-03-21

关于Q 函数的逼近

1. 线性逼近方法（Least Squares 类）

这些方法假设 Q 函数可以由状态-动作特征的线性组合表示：
$$
Q(s, a) = \phi(s, a)^T \theta
$$
其中：

  • $\phi(s, a)$ 是状态-动作的特征向量
  • $\theta$ 是权重参数

🔹 1.1 LSTD-Q（Least-Squares TD-Q Learning）

方法： 通过最小二乘法解线性方程：
$
(A - \gamma A’) \theta = b
$

  • 适用于小规模问题，计算成本较低。
  • 缺点： 计算 $( A^{-1})$ 可能不稳定（需要正则化）。

🔹 1.2 LSPI（Least-Squares Policy Iteration）

方法： 结合 LSTD-Q 和策略迭代，不断更新策略：

  1. 用 LSTD-Q 逼近 Q 函数
  2. 用贪心策略更新动作选择
  3. 循环迭代，直到收敛

特点：

  • 比 LSTD-Q 更稳定
  • 适用于策略迭代（Policy Iteration）类问题
  • 不适用于连续控制问题

🔹 1.3 Fitted-Q Iteration（FQI）

方法： 用监督学习的方法拟合 Q 函数：

  1. 采集大量 $(s, a, r, s’)$ 经验数据
  2. 用回归方法学习 $Q(s, a) \approx r + \gamma \max_{a’} Q(s’, a’)$
  3. 迭代更新，直到 Q 值收敛

特点：

  • 适用于高维状态空间
  • 可用 决策树、随机森林、神经网络等 逼近 Q 函数
  • 比 LSTD-Q 更灵活

2. 非线性逼近方法（深度学习）

当状态空间维度很高（如图像、文本），线性方法效果较差，我们通常采用 非线性方法 来逼近 Q 函数。

🔹 2.1 DQN（Deep Q-Network）

方法： 用 深度神经网络（DNN） 近似 Q 函数：
$
Q(s, a) \approx f_{\theta}(s, a)
$

  • 使用 神经网络 作为 Q 函数近似器
  • 通过 均方误差（MSE）最小化 损失：
    $
    L(\theta) = \mathbb{E} \left[ (r + \gamma \max_{a’} Q(s’, a’; \theta^-) - Q(s, a; \theta))^2 \right]
    $
  • 使用 目标网络（Target Network） 解决不稳定问题
  • 优点： 适用于高维离散动作空间
  • 缺点： 计算量大，训练难度高

🔹 2.2 DDPG（Deep Deterministic Policy Gradient）

  • 适用于连续控制任务（不像 DQN 只能用于离散动作）
  • 结合 策略梯度（Policy Gradient） 和 Q-learning
  • 主要用于机器人控制、自主驾驶等任务

🔹 2.3 SAC（Soft Actor-Critic）

  • 在 DDPG 基础上加入熵（Entropy），增加探索性
  • 适用于 高维、复杂任务
  • 比 DDPG 收敛更稳定

方法对比

方法 适用范围 是否非线性 优势 缺点
LSTD-Q 小规模问题 ❌ 线性 计算快，收敛快 不能处理高维数据
LSPI 小规模问题 ❌ 线性 结合策略迭代，稳定 需要大量样本
FQI 中等规模问题 ✅ 可用回归 适用于高维特征 训练慢
DQN 高维状态、离散动作 ✅ 神经网络 适用于复杂任务 训练不稳定，难以调参
DDPG 高维状态、连续动作 ✅ 神经网络 适用于机器人控制 训练不稳定
SAC 高维状态、连续动作 ✅ 神经网络 比 DDPG 更稳定 计算量大
